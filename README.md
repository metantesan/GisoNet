<h1 align="center">GisoNet</h1>

<p align="center">A local HTTPS proxy for development, with a desktop app, tray access, and background daemon mode.</p>

## Use it for

- mapping local apps to real-looking domains
- serving traffic over HTTPS on your machine
- keeping proxy control in one desktop app

## Install

Download the release for your platform and install it like any normal app.

- macOS: `.pkg`
- Debian and Ubuntu: `.deb`
- Linux: `.tar.xz`
- Arch Linux: package archive

On Linux, the installed command is `gisonet`.

## Start

1. Open `GisoNet` from your applications menu.
2. Add a route in the app.
3. Use the tray icon for quick access.
4. Start daemon mode if you want the proxy to keep running in the background.

## What you get

- desktop window for route management
- tray icon for quick access
- background daemon mode for always-on use
- system integration on Linux releases
